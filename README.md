## How to use

* Clone the repo `git clone https://github.com/scaint/vimfiles.git ~/vimfiles`
* Run `~/vimfiles/vimfiles install`

> Your existing `~/.vimrc` file and `~/.vim` dir will be renamed to `~/.vimrc_old` and `~/.vim_old`
